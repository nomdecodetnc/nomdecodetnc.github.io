Architecture LBL : organiser un programme Heidenhain

⏱ 8 min de lecture🏷 TNC 426 · LBL · 3 axes · 5 axes

Sur TNC 426, les labels LBL sont le mécanisme central d'organisation d'un programme. Un programme sans convention de labels est illisible — un CALL LBL 5 nu ne dit rien à l'opérateur qui reprend le programme 6 mois plus tard. Une convention structurée permet de lire le corps principal d'un programme comme une gamme d'opérations.

Principe général

Un programme Heidenhain bien structuré se décompose en trois zones distinctes :

  1. Le corps principal — la séquence des opérations, avec uniquement des CALL LBL commentés
  2. Les labels d'infrastructure — dégagements, annulation de plan, inclinaison de plan
  3. Les labels de pièce — positions, contours, cycles spécifiques à la pièce usinée

En lisant le corps principal, un opérateur doit comprendre la séquence complète sans entrer dans un seul label. Les CALL LBL commentés servent de table des matières lisible.

Convention des labels d'infrastructure

Ces labels sont identiques d'un programme à l'autre — ils constituent le socle réutilisable.

LBL 90 — Dégagement fin d'opération standard

* - LBL 90, DEGAGEMENTS Z M91
LBL 90
CALL LBL 100          ; annule le plan d'abord
L Z-1 R0 F MAX M91 M9  ; dégagement Z en coordonnées machine
LBL 0
STOP M2

LBL 91 — Dégagement complet fin de programme

* - LBL 91, DEGAGEMENTS Z,X,Y M91
LBL 91
L Z-1 R0 F MAX M91
L X-1 R0 F MAX M91
L Y-1 R0 F MAX M91
LBL 0
STOP M2

LBL 92 — Dégagement pour outil hors magasin

* - LBL 92, DEGAGEMENTS CHANGEMENT OUTIL HORS MAGASIN Z,Y,X M91
LBL 92
CALL LBL 100
L Z-1 R0 F MAX M91
L Y-521 R0 F MAX M91   ; position tableau de changement manuel
L X-1 R0 F MAX M91
LBL 0
STOP M2
💡 Les coordonnées M91 sont en coordonnées machine absolues — elles dépendent de la configuration physique de la DMU 70 et doivent être adaptées à chaque machine.

LBL 100 — Annulation de plan

Le label d'annulation de plan remet la machine en configuration de base : décalage de point zéro à zéro, plan d'usinage horizontal.

* - LBL 100, ANNULATION PLAN
LBL 100
CYCL DEF 7.0 POINT ZERO
CYCL DEF 7.1 X+0
CYCL DEF 7.2 Y+0
CYCL DEF 7.3 Z+0
CYCL DEF 19.0 PLAN D'USINAGE
CYCL DEF 19.1 A+0 B+0 C+0
CYCL DEF 19.0 PLAN D'USINAGE
CYCL DEF 19.1
LBL 0
STOP M2

LBL 101–108 — Inclinaisons de plan (5 axes)

Chaque plan incliné a son propre label, nommé avec les paramètres d'inclinaison dans le commentaire. C'est le point le plus important pour la lisibilité en 5 axes.

* - LBL 103, INCLINAISON PLAN : B-90, DEC Y-16.5 Z-20
LBL 103
CYCL DEF 7.0 POINT ZERO
CYCL DEF 7.1 Y-16.5
CYCL DEF 7.2 Z-20
CYCL DEF 19.0 PLAN D'USINAGE
CYCL DEF 19.1 B-90
L B-90 C+0 R0 F MAX
L B+Q121 C+Q122 R0 F MAX
LBL 0
STOP M2

Convention d'appel — comment lire le corps d'un programme

Chaque CALL LBL dans le corps principal doit être commenté avec ce qu'il fait, pas avec son numéro. Le numéro est déjà visible.

À éviterÀ utiliser
CALL LBL 103 CALL LBL 103 ; INCLINAISON PLAN : B-90, DEC Y-16.5 Z-20
CALL LBL 90 CALL LBL 90 ; ANNULATION PLAN, DEGAGEMENTS Z M91 M9
CALL LBL 28 CALL LBL 28 ; X+76.107 Y-36, Z+2 M13 M7

Différences entre 3 axes et 5 axes

En 3 axes

Il n'y a pas de plan d'usinage incliné, pas de CYCL 19, pas de gestion des axes B et C. Le label d'annulation de plan se réduit à un simple dégagement Z.

* - LBL 90, DEGAGEMENT Z
LBL 90
L Z+100 R0 F MAX M9
LBL 0

Les labels d'inclinaison (LBL 101–108) n'existent pas — ils sont remplacés par de simples positionnements XY.

En 5 axes

Chaque changement de face nécessite :

  1. Un retour en position neutre (CALL LBL 90 ou 100)
  2. Un positionnement B+Q121 C+Q122 (axes rotatifs en position de changement d'outil)
  3. Un appel du label d'inclinaison avec ses paramètres DEC et angle B
  4. Un positionnement en X, Y, Z sécurisé avant d'aller en zone de coupe
CALL LBL 90 ; ANNULATION PLAN, DEGAGEMENTS Z M91 M9

L B+Q121 C+Q122 R0 F MAX
CALL LBL 103 ; INCLINAISON PLAN : B-90, DEC Y-16.5 Z-20

L X+50 Y+0 F MAX  ; positionnement sécurisé hors pièce
L Z+2 F MAX
; ... opérations ...
CALL LBL 90 ; ANNULATION PLAN, DEGAGEMENTS Z M91 M9
⚠️ En 5 axes, ne jamais appeler un label d'inclinaison sans avoir annulé le plan précédent — le TNC 426 peut calculer une trajectoire inattendue si un plan est déjà actif.

Labels de pièce — convention de numérotation

Les labels spécifiques à la pièce (contours, positions, cycles) sont numérotés après les labels d'infrastructure. Une convention courante :

  • LBL 1–9 : contours géométriques principaux
  • LBL 10–29 : positions et cycles d'ébauche
  • LBL 30–39 : contours et cycles de finition
  • LBL 25 : réservé au surfaçage (par convention)
  • LBL 90–108 : infrastructure (dégagements, plans)
💡 La numérotation exacte n'est pas figée — l'essentiel est d'avoir une convention cohérente et documentée, appliquée sur tous les programmes de l'atelier.
