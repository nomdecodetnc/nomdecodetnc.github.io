Sécurité programme : différences fondamentales entre 3 axes et 5 axes

⏱ 7 min de lecture🏷 5 axes · Dégagements · TNC 426 · Plan incliné

Les labels de sécurité — dégagements, annulation de plan, positionnements entre opérations — ne se construisent pas de la même façon en 3 axes et en 5 axes. Comprendre cette différence est fondamental pour programmer sans risque de collision.

En 3 axes : la sécurité est simple

En 3 axes, il n'y a pas de plan incliné. L'axe Z est toujours perpendiculaire à la table. Un dégagement Z est toujours sûr — monter en Z écarte systématiquement l'outil de la pièce.

* - Dégagement fin d'opération 3 axes
L Z+100 R0 F MAX M9  ; monte à Z+100 — toujours sûr

Entre deux opérations, un positionnement XY après le dégagement Z est sans risque — la pièce est en dessous et l'outil est haut.

En 5 axes : la table s'incline, le Z n'est plus "haut"

Quand la table est inclinée à B-90° (face latérale usinée), le Z machine n'est plus perpendiculaire à la surface usinée. Un simple L Z+100 peut envoyer l'outil vers la table au lieu de s'en éloigner — selon l'orientation en cours.

C'est pour ça que les dégagements en 5 axes utilisent M91 (coordonnées machine absolues) plutôt que des coordonnées de pièce.

* - LBL 90, DEGAGEMENTS Z M91 — 5 axes
LBL 90
CALL LBL 100       ; annule le plan d'usinage d'abord
L Z-1 R0 F MAX M91 M9  ; Z en coordonnées machine — toujours sûr quelle que soit l'inclinaison
LBL 0
⚠️ Ne jamais faire un dégagement Z sans avoir annulé le plan d'usinage (CALL LBL 100) en 5 axes. Le TNC 426 calcule la trajectoire dans le repère du plan actif — une collision est possible si le plan est encore incliné.

La séquence obligatoire entre deux opérations en 5 axes

; --- Fin de l'opération sur face B-90 ---
CALL LBL 90   ; ANNULATION PLAN, DEGAGEMENTS Z M91 M9

; --- Positionnement axes rotatifs ---
L B+Q121 C+Q122 R0 F MAX  ; position de changement d'outil ou de repositionnement

; --- Inclinaison pour la face suivante ---
CALL LBL 104  ; INCLINAISON PLAN : B+90, DEC X+79.107 Y-46.25 Z-10

; --- Positionnement sécurisé hors pièce avant d'aller en zone de coupe ---
L X+50 Y+0 F MAX
L Z+2 F MAX

L'ordre est immuable : annulation → positionnement rotatif → inclinaison → positionnement XY → descente Z. Inverser les étapes crée un risque de collision.

Q121 et Q122 : les paramètres de position de changement d'outil

Q121 (angle B) et Q122 (angle C) sont des paramètres machine définis par le constructeur. Ils définissent la position des axes rotatifs pour le changement d'outil — position neutre, hors zone de bridage, accessible au changeur automatique.

L B+Q121 C+Q122 R0 F MAX  ; position neutre de la machine

Ces valeurs sont lues depuis les paramètres machine et ne doivent pas être codées en dur dans le programme — si la machine change, les Q121/Q122 s'adaptent automatiquement.

Cas particulier : CYCL 8 IMAGE MIROIR en OP2

En OP2 avec la pièce retournée dans l'étau, CYCL DEF 8 IMAGE MIROIR X permet d'utiliser les mêmes labels de positions sans recalculer toutes les coordonnées. Le miroir s'applique à toutes les opérations jusqu'à l'annulation avec CYCL DEF 8.1 (sans paramètre).

CYCL DEF 8.0 IMAGE MIROIR
CYCL DEF 8.1 X              ; miroir en X activé
; ... opérations ...
CYCL DEF 8.0 IMAGE MIROIR
CYCL DEF 8.1                ; annulation du miroir
💡 CYCL 8 est particulièrement utile en OP2 pour les perçages et taraudages qui ont les mêmes positions relatives par rapport à la pièce, mais dans une orientation X inversée après retournement.
